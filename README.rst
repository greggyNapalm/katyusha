Katyusha: Load testing tool
===========================

Installation
------------
Use `gvm <https://github.com/moovweb/gvm/>`_ if needed

.. code-block:: bash

    $ go get github.com/greggyNapalm/katyusha/katyusha
    # $ katyusha --version
    Katyusha load tool v.0.0.1


Requirements
------------

* GNU Linux
* Golang >= 1.2
